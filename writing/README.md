# Writing Skills  
以下为写作与研究类 Claude 技能简要说明：  
- **Skill Creator**：元技能，快速生成新技能模板。  
- **Content Research Writer**：自动检索资料和润色文章，提高内容质量。  
- **Article Extractor**：提取网页正文文本。  
- **Brainstorming**：结构化头脑风暴辅助完善想法。  
- **Meeting Insights Analyzer**：分析会议记录，提供沟通洞察。  
- **YouTube Transcript**：获取视频字幕并生成摘要。  
- **Family History Research**：指导家族史研究项目。

# Design Skills  
以下为设计创意类 Claude 技能简要说明：  
- **Canvas Design**：根据文本生成平面设计稿。  
- **Imagen**：调用外部图像生成 API 生成插图或界面。  
- **Image Enhancer**：提升图片清晰度和分辨率。  
- **Theme Factory**：为文档、幻灯片套用预设主题样式。  
- **Video Downloader**：下载网络视频用于离线使用或剪辑。  
- **YouTube Transcript**：提取视频字幕并生成摘要。  
- **Algorithmic Art**（候选）：使用 p5.js 生成抽象艺术作品，展示创意潜力。  
- **Slack GIF Creator**（候选）：生成 Slack 动图，趣味性强但适用度有限。

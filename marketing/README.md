# Marketing Skills  

以下为市场营销类 Claude 技能简要说明：  

- **Lead Research Assistant**：自动化挖掘潜在销售线索并生成外联策略。  
- **Internal Comms**：根据要点生成规范化的内部通讯稿、周报等。  
- **Competitive Ads Extractor**：抓取竞品广告素材并分析创意。  
- **Domain Name Brainstormer**：根据关键词生成创意域名并检测可注册性。  
- **Brand Guidelines**（候选）：管理品牌色彩与字体，确保输出符合品牌规范，但许可证受限，仅供参考。
